Search & Visualise

  1. Start by simply typing the name of a field from your data to produce a KPI.
  2. If nothing happens the chosen field is probably not considered as measurable but you can use words and phrases like 'sum' or 'how many' in front of the field name to be more precise.
  3. You can include multiple fields to start producing more sophisticated visualisations and if you want something specific, like a line chart, simply add 'linechart' to your search terms.
  4. If you want to filter the data you can add field values and if there are any ambiguities you can tell the search what you meant.
